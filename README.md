# consumption-tax-calculator
consumption tax calculator

```sh
❯ python3 main.py
=== 消費税申告（簡易課税）2025年分（令和7年分）/ 第5種(50%) / 10%のみ ===
入力：所得税の確定申告書 第一表「収入金額等（事業）」の金額（税込）を想定します。
（前提：非課税・不課税・売上以外収入・8%売上・調整事項なし）

2025年の事業収入（税込）を入力してください: 15800000
基準期間（原則：2023年）の課税売上高（税抜）を入力してください（例：12315000）: 12800000
中間納付（国税）があれば入力（なければ空欄/0）: 449000
中間納付（地方）があれば入力（なければ空欄/0）: 120200
```
